# ROLA

## For running locally

`npm i`

### Start the dev server

`npm run dev`

### Build the project

`npm run build`

### Start built project

`npm start`

## For running Docker Containers

_You'll need docker installed on your machine to run this!_

### Build the image

`docker-compose build`

### Start the dev server

`make up`

### Build and start dev server

`make up-build`

### Stream logs

`make logs`

### Stop the server

`make down`

### Build and start production build

`make up-prod`
